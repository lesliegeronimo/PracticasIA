## Ejecución
Para compilar y ejecutar este código desde la terminal, se recomiendan los comandos siguientes:

```
javac -d ./classes -cp lib/core.jar:. gatos/*.java
java -cp ./classes:lib/core.jar gatos.Gatos
```
